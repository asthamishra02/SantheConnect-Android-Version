# SantheConnect – Android Version

SantheConnect is an Android application built to promote Karnataka’s local culture, traditional markets (Santhes), authentic food points, and small-scale hospitality providers that usually do not have a digital presence.

The app acts as a discovery platform for tourists looking for authentic local experiences while helping local vendors gain digital visibility.

---

# Problem Statement

Tourists visiting heritage towns often struggle to discover:

* Weekly local markets (Santhes)
* Authentic local eateries
* Traditional craft stalls
* Home-style food points

Most of these places are not available on mainstream travel platforms.

SantheConnect bridges this gap using Android + Firebase + Google Maps integration.
